Pokemon

Show stats of each made pokemon (base damage, base defense, base stamina, attack, defense, stamina, type1, type2, MaxHP and CP (combat power)).
Every pokemon has a few quick moves (fast attack) and main moves (charging attack) and it depends on the pokemon but for every round 1 pokemon has got only 1 quick move and 1 or 2 charging attacks. Each fast attack or charging attack has its base damage, if a Pokemon's type(type1 or type2) matches the move's type (quick move or main move) than his damage amplifies and it is called STAB (Same-type attack bonus). So, if that is the case, Pokemon's boost in that specific move type is increased by 20%.
